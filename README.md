## Preliminaries 
Create a venv:
1. ``` pip install virtualenv ```
2. ``` python -m venv env ```
3. ``` source env/bin/activate ```

Download requirements.txt:
1. ```pip install -r /path/to/requirements.txt```

 ## Files

- ```bf_env.py``` is an OpenAI gym environment
- ```sarsa.py``` is uncompleted
